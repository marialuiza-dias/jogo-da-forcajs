Regras do Jogo 
  O jogador inicia o jogo informando seu nome e escolhendo uma categoria de palavras.
  O jogador tem um limite máximo de 6 erros para tentar adivinhar a palavra secreta.
* Sistema de Pontuação: 
  Cada letra acertada adiciona 10 pontos
  Cada letra errada subtrai 5 pontos
  Vencer a rodada garante um bônus de 50 pontos
  A pontuação final nunca fica negativa (mínimo é 0).
* Sistema de Dicas:
  O jogador pode digitar a palavra `DICA` a qualquer momento para receber uma pista sobre a palavra secreta.
* Penalidade: 
  O uso da dica custa 1 tentativa (contabiliza +1 erro no contador).
  O jogo ignora diferenças entre letras maiúsculas e minúsculas e avisa se uma letra já foi tentada, sem penalizar o jogador.

Como Jogar
 1. Assim que o jogo iniciar, digite o seu nome no terminal.
 2. Escolha uma das categorias disponíveis digitando o número correspondente.
 3. Chute uma letra por rodada ou digite `DICA` caso precise de ajuda.
 4. Ao final da rodada, o jogo mostrará o seu resultado e a pontuação. Digite `Y` para jogar novamente ou `N` para sair.

Como Executar
 Ter o Node.js instalado em seu computador.

1. Abra o terminal na pasta do projeto.
2. Instale as dependências necessárias executando:
 bash
 npm install
